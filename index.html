Image Restoration using Deep Learning

Image restoration tasks are often time-consuming and require manual intervention. To automate these tasks, I developed a deep learning project to produce high-quality images using neural networks to apply various image restoration techniques. Used PyTorch to train neural network models for various image restoration tasks. I also used Pillow (PIL) Library to process images and NumPy, Pandas to handle image data. To serve the project live on a server, I created a Flask app that used the previously trained models to process images and produce the result. The Flask app was wrapped in UWSGI which connected to NGINX. For the front-end, I used JavaScript, HTML, and CSS to provide input images to be uploaded to the server and display the output images served by the server. To optimize the system, I implemented GPU acceleration to speed up the processing of large images. I also ensured that the Flask app was optimized for high concurrency to handle multiple requests simultaneously.
